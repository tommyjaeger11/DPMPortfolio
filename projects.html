A couple of my recent projects:

These are a few of my recent projects to showcase my skills and what I can apply to my professional career.

  • Casey's Darn Good Coffee Advertising Project

    In this project our group designed and impolemneted an Oracle Cloud relational database, conducted SQL queries and statistical analysis to measure promotional effectiveness, identified trends in customer retention, consumer spending habits, and promotional success. We also developed interactive KPIs to present revenue impact, regional market differences, ROI dashboard of advertising campagins, and recomendations with actionable insights for senior analysts of Casey's General Store.

  • Microsoft Grade ETL Data Pipeline

    In this project I built a full ETL pipeline using Microsoft Azure. Started with setting up the Infrastructure using Azure Containers, SQL Database, SQL Server, and Azure Data Factory. I then created and triggered a ETL Pipeline to load data into Azure and to PowerBI for Visualization. Finally, I created a PowerBI dashboard to visualize the data and presented it to the client.

  • Pyspark Machine Learning Internet Serivce Provider Customer Churn Predictions

    Conducted predictive modeling on ISP Churn data using Pyspark in Databricks, applying logistic regression, random forest, and linear regression modles to identify churn liklihood, risks and key behavior drivers. We then compared these model performance metrics using AUC, accuracy and RSME to evaluate predictive power and guide marketing and operation strategy recommendations.
